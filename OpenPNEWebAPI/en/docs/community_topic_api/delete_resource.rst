=============================
Community Topic::Delete Entry
=============================

Request
=======

The Way to Request
------------------

Do DELETE to Member URI.

::

  DELETE /api.php/feeds/communityTopic/{community_topic_id} HTTP/1.1
  Host: example.com

Request Parameter
-----------------

None

Response
========

Response Header
---------------

Returns HTTP Status Code 200.

::

  HTTP/1.1 200 Ok

Response Body
-------------

None
