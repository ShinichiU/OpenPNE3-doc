日記::エントリの削除
====================

リクエスト
==========

リクエストの方法
----------------

メンバ URI に対して DELETE リクエストをおこなう。

::

  DELETE /api.php/feeds/diary/{diary_id} HTTP/1.1
  Host: example.com

リクエストパラメータ
--------------------

なし

レスポンス
==========

レスポンスヘッダ
----------------

HTTPステータスコード 200 を返す。

::

  HTTP/1.1 200 Ok
