symfony について
================

OpenPNE3 は symfony をベースに作られています。そのため、 symfony についてある程度知っておく必要があるでしょう。

symfony は PHP のウェブアプリケーション・フレームワークで、ウェブアプリケーションを構築するのに有用な数多くの機能を備えています。

symfony は大規模なサイトでも用いられています。たとえば、 Delicious や Yahoo! Bookmarks、 Dailymotion などです。symfony はこういった大規模なサイトも視野に入れて作られています。

また symfony は高い拡張性を備えています。 symfony は素晴らしいプラグインシステムを備えています。プラグインはこのシステムによって symfony に多くの機能を加えることができます。たとえそれが ORM であったとしてもです。

OpenPNE3 は symfony のもたらす拡張性の上に成り立っていると言っても過言ではないでしょう。

OpenPNE3 のインストール
=======================

動作要件
--------

OpenPNE3 は symfony をベースにしており、 Doctrine を用いています。そのため OpenPNE の要求はこれらに影響を受けています。

OpenPNE3 は PHP 5.2.3 以上を要求しています。また PHP のいくつかの拡張が有効になっている必要があります。たとえば、 mbstring や xml、 pcre、 pdo （さらに DBMS のドライバが必要）などです。

次に示すコマンドを叩くことで環境情報を知ることができます::

  $ php -v
  $ php -m

.. note::

  もし動作要件に関する完全な情報が知りたい場合、 OpenPNE3 のディレクトリに入っているインストールドキュメントを参照してください。
  
  なぜなら動作要件の細部は、メジャーリリースによって変わっていく可能性があるからです。
